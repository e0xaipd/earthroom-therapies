
# Earthroom Therapies (Static Site)

A nature‑inspired, sage‑green website for a therapy practice. Built as a simple static site: HTML + CSS + a tiny bit of JS.

## Quick Start
Open `index.html` in your browser, or deploy the whole folder to any static host (GitHub Pages, Netlify, Azure Static Web Apps, etc.).

## Customize
- Replace images in `assets/img/` with your own photography. Use the `.hero .bg` inline styles to point to new files.
- Update copy on each page as needed.
- Colors live in `assets/css/style.css` `:root` variables.

## Notes
- The contact form is a non‑functional demo. Connect it to your backend or a form service before using in production.
